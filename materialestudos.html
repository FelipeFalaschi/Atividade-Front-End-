Material de Estudo: VisualG

O VisualG é uma ferramenta utilizada para aprender lógica de programação. Ele permite criar algoritmos utilizando pseudocódigo de forma simples e didática, sendo muito utilizado por iniciantes na programação.

O que é um algoritmo?

Um algoritmo é uma sequência de passos utilizada para resolver um problema. Na programação, os algoritmos ajudam a organizar o raciocínio antes de escrever o código em uma linguagem de programação.

Exemplo de algoritmo no VisualG

algoritmo "exemplo"

var
numero: inteiro

inicio
escreva("Digite um número: ")
leia(numero)

escreva("O número digitado foi: ", numero)

fimalgoritmo

Principais comandos do VisualG

  • escreva() → mostra informações na tela
  • leia() → recebe dados do usuário
  • se / então / senao → estrutura condicional
  • para / enquanto → estruturas de repetição
